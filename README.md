HICapacity Devops Club
======================

This is a vagrant machine that will allow you to play with the [runc](https://github.com/opencontainers/runc)
cli tool from https://www.opencontainers.org/.

People present at the meeting:

  * McKay Davis
  * Ryan Mohr
  * Stephan Fitzpatrick
  * Chae Lee
  * Ryan Kanno

Install
-------

  * `vagrant up`

Run
---

  * `vagrant ssh`
  * `cd runc`
  * `sudo runc`
